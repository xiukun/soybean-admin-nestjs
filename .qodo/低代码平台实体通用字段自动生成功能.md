# 低代码平台实体通用字段自动生成功能

## Core Features

- 实体创建时自动添加通用字段

- 实体字段管理和验证

- 项目代码生成时自动创建数据库表

- 通用字段类型定义和约束管理

- 实体创建API增强

## Design

基于NestJS + Prisma + PostgreSQL的微服务架构，使用装饰器和拦截器实现自动字段添加，利用Prisma Schema生成器动态创建数据库表结构
