# 低代码平台项目管理系统

## Core Features

- 项目激活停用管理

- 实时部署监控

- 代码生成集成

- 模板管理

- 导入导出功能

- 协作功能

- 性能优化

- 国际化支持

- 模块化测试

## Tech Stack

{
  "Web": {
    "arch": "vue",
    "component": "naive-ui"
  },
  "Backend": "NestJS + Prisma + PostgreSQL",
  "Container": "Docker微服务",
  "State": "Pinia状态管理",
  "Testing": "Vitest测试框架"
}

## Design

现代化管理界面设计，白色和浅灰色主色调，蓝色品牌色，卡片式布局和响应式网格设计
